# Opus 4.7 发布，按次计费版暂不支持

> 发布日期：2026-04-17 &nbsp;|&nbsp; 影响范围：按次计费版用户

## 概要

Cursor 官方已上线最新的 **Opus 4.7** 模型。但由于该模型**强制开启 Max Mode**，按次计费版用户**暂时无法使用**。按量计费版用户不受影响，可正常使用。

## 为什么按次计费版不能用？

Opus 4.7 属于 **Max Only** 模型——官方要求必须在 Max Mode 下才能调用。而 Max Mode 会将计费方式从**按次**切换为**按量计费**，对按次计费版用户来说极其危险：

- 按量计费模式下，官方换算规则为 **1 次 request = 0.04 美元**
- 上下文较大时，一次对话可能消耗数美元，折算为**几百次额度瞬间扣完**
- 已有用户仅一次对话便将 500 次额度**全部耗尽**

::: danger 额度无法恢复
额度一旦被官方扣除，无法通过任何方式恢复。向官方申诉不仅不会返还额度，甚至可能导致账号资格被取消。
:::

简单来说：**Opus 4.7 = 强制 Max Mode = 按量计费 = 按次版额度瞬间归零**。这与之前禁止手动开启 Max Mode 是同一个道理。

## 各版本影响

| 版本 | 能否使用 Opus 4.7 | 说明 |
| --- | --- | --- |
| **按量计费版** | ✅ 可以 | 本身就是按量扣费，Max Mode 不影响计费逻辑 |
| **按次计费版** | ❌ 不可以 | 强制 Max Mode 会导致额度被快速消耗 |

## 按次计费版用户怎么办？

1. **继续使用其他模型** Opus 4.6 等非 Max Only 模型仍可正常使用
2. **不要尝试强行选择 Opus 4.7**：即使模型列表中出现，选择后会触发 Max Mode 计费
3. **考虑升级按量计费版**：如果对 Opus 4.7 有刚需，可联系管理员了解按量计费版详情

::: warning 提醒
所有标注 **MAX Only** 的模型，按次计费版用户均不可使用。这不仅限于 Opus 4.7，未来官方新增的 Max Only 模型同样适用此规则。
:::
