# 如何查看用量（按量计费版）

按量计费版的额度以**美元**为单位，扣费顺序为：**基础 $20 → 赠送额度（浮动）→ 额外额度（按需计费）**。

::: warning 注意
额外额度部分近期显示可能不准确，请以**三项加总**后的总计为准。
:::

## 方法一：导出 CSV 统计

1. 登录 [Cursor 官网](https://cursor.com)
2. 进入 **Settings → Usage** 页面
3. 点击 **Export CSV** 下载用量明细
4. 打开 CSV 文件，统计 **Cost 列的总和**即为您的实际累计用量

## 方法二：在编辑器内查看用量摘要

通过编辑器内置的 Usage Summary 功能，可以直接查看基础 / 赠送 / 额外三部分各自用了多少。

### 开启步骤

1. 点击编辑器**右上角齿轮图标**，进入设置
2. 选择 **Cursor Settings**
3. 找到 **Agents → Usage Summary**
4. 将其设置为 **Always**

开启后，在对话框底部即可看到用量摘要，分别显示**基础额度**、**赠送额度**、**额外额度**的使用情况，三项相加即为您的总计用量。
::: warning 20260402 updated
赠送目前和额外可能会搞混，可能存在赠送100+(正常赠送在60以内)，额外0，但是实际上总额已达标提示上限的情况。
:::
## 关于赠送额度

官方会在基础用量之外提供额外赠送额度。赠送额度的多少可能因套餐和时期不同而有所浮动。

以下为官方对赠送额度的说明：

![官方赠送额度说明](/images/bonus-usage-explanation.png)

> 官方原文要点：~~各套餐~~均包含基础 API Agent 用量额度 + 额外赠送用量。官方会尽量在保证基础用量之外提供更多额外赠送额度。由于不同模型的 API 成本不同，实际消耗速度取决于您选择的模型和上下文大小。（Pro等个人版系列不再包含赠送，而是改为单独用量池）

## 扣费顺序说明

| 顺序 | 额度类型 | 说明 |
| --- | --- | --- |
| 1 | 基础额度（$20） | 所有套餐统一的基础部分，最先扣除 |
| 2 | 赠送额度（浮动） | 官方赠送部分，基础用完后扣除 |
| 3 | 额外额度（按需） | 属于按需计费（On-Demand），最后扣除 |

套餐标注的总额度 = 基础额度 + 赠送额度。例如 60 美元套餐即 $20 基础 + $40 赠送，质保按 $60 计算。
