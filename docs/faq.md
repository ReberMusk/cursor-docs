# 常见问题

## 我怎么选套餐？

两个版本各有优势，需要根据使用习惯来选，不存在哪个版本绝对更好。

### 适合选按次计费版的情况

- 经常使用 Opus、Codex、Gemini 等**高级模型**
- 单次对话的**上下文较大**（长文件、多文件引用等）
- 使用高级模型 + 大上下文时，按量计费下一次对话可能消耗数美元，而按次计费固定只扣 1\~2 次，**成本优势明显**

### 适合选按量计费版的情况

- 对高级模型需求不高，日常以中低成本模型为主
- 单次对话上下文较小，按量计费下每次消耗不多
- 希望使用 Auto / Composer 等功能，或需要更灵活的模型组合

### 简单判断

| 使用习惯 | 推荐版本 |
| --- | --- |
| 高频使用高级模型 + 大上下文 | **按次计费** |
| 中低频使用 + 上下文不大 | **按量计费** |
| 重度依赖 Auto / Composer | **按量计费** |

详细套餐说明请参阅 [按量计费版](/plans/team-metered) 和 [按次计费版](/plans/team-request)。

## 团队版是什么？和 Pro 有什么区别？

团队版本质上也是**独享账号**，不是多人共用。它对应官网 **40 美元 / 席位**的团队套餐，只是通过更灵活的方式开通，更适合对 API 模型通用额度需求较高的用户。

主要区别在额度结构：

- **Pro**：20 美元基础额度 + 20 美元透支额度，另有独立的 Auto / Composer 用量池。更适合经常使用 Auto / Composer 的用户
- **团队版（按量计费）**：通用额度池，所有官方支持的模型统一计费，没有单独拆分 Auto 系列额度。一般 60 美元起，规格更灵活

详细对比请参阅 [按量计费版介绍](/plans/team-metered#与-pro-的区别)。

## 发的是成品账号吗？是独享的吗？可以代充吗？

**是成品账号，也是独享账号。**

团队版都是预先分配好的新账号，因此暂**不支持在您已有的老账号上代充**。

账号到手后可自行修改密码，改完后就是您独立使用，和自行注册的账号没有区别。登录新账号**不会影响**您原有账号的聊天记录，原账号的内容仍然保留。

## 额度能用多久？能换算多少 Token？

无法提前给出固定值。官方根据您**实际使用的模型、上下文大小、调用情况**动态扣费，具体能用多久取决于您的使用习惯。

额度统一按**美元**计算，不同模型价格差异很大，换算出来的 Token 数量也不同。例如：

- 某些模型 1 美元 ≈ 10 万 Token
- 另一些模型 1 美元 ≈ 100 万 Token

## 账号质保多久？

默认**全程质保 1 个月**。

质保期内出现**掉会员**或**封号**的情况，会按照**剩余天数折算退款**。

::: warning 注意
额度正常用完不属于质保范围。
:::

## 能用哪些模型？可以多设备登录吗？

**官方支持的模型都可以使用**。部分海外模型需要自备可用网络环境。

关于多设备登录：账号是独立账号，同一人在公司和家里两台设备登录使用一般没有问题。但为稳妥起见，**不建议多人共用**，也尽量**避免同时在线使用**。具体以官网规则为准。
