# 如何切换账号

切换账号的核心逻辑：**先在浏览器和编辑器中分别退出旧账号，再从浏览器登录新账号，最后在编辑器中完成授权。**

## 切换步骤

### 第一步：在浏览器退出旧账号

1. 在**默认浏览器**中打开 [cursor.com](https://cursor.com)
2. 点击左上角头像旁边的选项，退出当前登录的账号

### 第二步：在浏览器登录新账号

退出后，直接在浏览器中**登录新账号**。

### 第三步：在编辑器退出旧账号

1. 打开编辑器设置，进入 **General**（通用）页面
2. 滚动到**最底部**，点击**退出账号**

### 第四步：在编辑器完成授权

1. 在编辑器**顶部**点击**登录**
2. 会自动跳转到默认浏览器，点击**同意授权**即可完成登录

## 关于聊天记录

切换账号**不会影响聊天记录**。切换到新账号后，之前的对话仍然保留，可以继续上次的对话。

::: warning 注意备份 Rules 配置
切换账号可能会影响之前配置的 **Rules**（自定义规则）。如果您之前配置过 Rules，建议在切换前**自行备份**。没有配置过的可以忽略。
:::

## 常见问题

### 点击授权没有反应？

可能的原因：

1. **编辑器网络不通**：检查编辑器是否能正常联网，确认代理 / VPN 设置是否覆盖到了编辑器
2. **编辑器缓存错乱**：需要**清除缓存后卸载重装**编辑器，然后重新执行上述切换步骤
