# 进度条显示不对怎么办？

::: warning 这是一个临时 Bug，不是长期问题
该问题由 2026 年 4 月 15 日前后的官方代码变更引起，官方已确认为 Bug 并将在后续修复。
:::

如果您看到仪表盘进度条已满（如 500 / 500），但实际并未用完额度，这是官方设置页面的**显示错误**，不影响实际使用。

**您的套餐限额没有改变，收费方式也没有改变。**

## 怎么办？

1. **不用慌**，正常使用即可，显示异常不影响实际额度
2. 如需确认真实用量，请参阅 [如何查看用量（按次计费版）](/guide/check-usage)
3. 完整的问题说明、截图对比和官方回复请查看公告：
