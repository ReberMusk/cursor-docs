# 找不到Opus /Opus 没思考没Max？

## 找不到 Opus 模型

Opus 系列模型的**地区限制较为严格**。如果在模型列表中看不到 Opus，通常是因为当前网络出口 IP 所在地区不被支持。

### 不支持的地区

中国大陆、中国香港、中国台湾等地区均**不在 Opus 的可用范围内**。

### 解决方法

1. 确认编辑器的**出口 IP** 位于模型支持的地区（如美国、日本等）
2. 切换网络环境后，**刷新模型列表**即可看到 Opus

::: tip 提示
判断标准是编辑器实际的出口 IP 地址，而非系统代理设置。请确保 Cursor 的网络请求确实经过了支持地区的节点。
:::

## 找不到思考模式（Thinking）

新版 Cursor 调整了 UI 布局，Thinking 模式的开关位置发生了变化。它不再出现在模型名称旁边，而是移到了对话框内的独立选项区域。

### 开启方法

1. 在对话框底部的**模型选择器**中选择需要的模型
2. 点击模型选择器旁边的 **Options**（或 `...` More 按钮）
3. 在弹出的面板中即可看到 **Thinking** 开关，以及 Context（200K / 1M）和 Effort（Low / Medium / High / Max）等选项

![新版 UI 中的模型选项与 Thinking 开关位置](/images/model-options-ui.png)

::: warning 按次计费用户注意
Options 面板中的 **Fast**、**1M Context**、**Max Effort** 均与 Max Mode 绑定，开启后会切换为按量计费。按次计费用户请勿开启这些选项，详见 [按次计费版 - 禁止操作](/plans/team-request#禁止操作)。
:::
