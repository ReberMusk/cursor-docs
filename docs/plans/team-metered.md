# 团队版 - 按量计费

团队版对应官网 **40 美元 / 席位**的团队套餐，发放的是**独享成品账号**，与自行注册的账号使用体验一致。适合对 API 模型通用额度需求较高的用户。

之所以叫“团队版”，是因为它对应的是官网的团队套餐权益。您可以理解为通过拼团方式开通该版本，**每位用户都是独立使用、互不影响**，同时都能享受团队版对应的权益和额度。

## 产品概述

- 账号类型：**独享成品账号**，非多人共用
- 对应官网套餐：Business / Team（40 美元 / 席位）
- 额度类型：**通用美元额度**，所有官方支持的模型均可正常计费使用
- 起步规格：**累计 60 美元额度起**（含官方赠送），规格灵活可选

::: warning 关于登录后只看到 $20
登录后您看到的 $20 是**基础额度**，这是正常的。官方赠送的额度不会预先显示，而是在基础 $20 用完后**自动开始抵扣并显示**。详见下方 [赠送额度何时显示？](#赠送额度何时显示) 章节。
:::

## 额度结构

所有套餐的基础额度均为 **20 美元**，此外官方会赠送一部分额度。套餐标注的金额为**基础额度 + 赠送额度**的总和。

以 60\~80 美元套餐为例：

| 组成部分 | 金额 |
| --- | --- |
| 基础额度 | 20 美元 |
| 赠送额度 | 40\~60 美元 |
| **标注总额度** | **60\~80 美元** |

> 质保按标注额度计算。例如 60 美元套餐，即保障您至少可使用到 60 美元。

**扣费顺序**：标配额度 → 赠送额度 → 额外（按需计费）。其中额外部分属于按需计费，会在最后扣除。

### 赠送额度何时显示？

赠送额度不会一开始就显示。当您的 **基础 $20 额度用完后**，赠送额度会自动开始抵扣，并在界面上显示出来。

如下图所示，基础 $20 已用完，赠送额度已开始抵扣 $0.29，显示为 `+$0.29 free usage on us`。赠送额度会持续抵扣，直到用完为止。

## 与 Pro 的区别

| 对比项 | Pro | 团队版（按量计费） |
| --- | --- | --- |
| 额度结构 | 20 美元基础 + 20 美元透支，另有独立的 Auto/Composer 用量池 | 通用额度池，不拆分 Auto 系列 |
| 适合场景 | 经常使用 Auto / Composer 的用户 | 对 API 模型通用额度需求更高的用户 |
| Auto/Composer | 有单独额度池 | 统一从通用额度扣费 |
| 起步额度 | 约 40 美元（含透支） | 60 美元起 |

::: info 说明
Pro 标注的 40 美元额度，实际由 20 美元基础额度 + 20 美元透支额度组成。如无透支部分，实际仅有 20 美元基础额度。
:::

## 交付说明

- 发放的是**成品独享账号**，到手后可自行修改密码
- 暂**不支持在您已有的老账号上代充**
- 登录新账号**不会影响**您原有账号的聊天记录，原账号内容仍然保留

## 使用须知

- **模型范围**：官方支持的所有模型均可使用。部分海外模型需自备可用网络环境
- **多设备登录**：同一人在公司和家里两台设备登录一般没有问题。但不建议多人共用，也尽量避免同时在多台设备上使用。具体以官网规则为准
- **额度消耗**：按实际使用的模型、上下文大小、调用情况动态扣费。不同模型价格差异大，无法提前给出固定 Token 换算值

## 质保政策

- 默认**全程质保 1 个月**
- 质保期内出现**掉会员**或**封号**，按剩余天数或剩余额度折算退款
- **额度正常用完不属于质保范围**
